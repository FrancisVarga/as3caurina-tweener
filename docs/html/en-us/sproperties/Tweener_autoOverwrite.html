Tweener Documentation

Property summary

autoOverwrite

Availability

AS2 and AS3.

Usage

Tweener.autoOverwrite = value;

Description

value:Boolean — A true or false value that denote whether new tweens created with addTween should overwrite other tweens already applied to the same object at the same time to the same properties or not. The default value for autoOverwrite is true.

This property defines the default tween creation behavior. Notice that the value of this property is ignored when creating new tweens if the overwrite parameter is used on new tweens.

Examples

// Default behavior: a tween is overwritten
Tweener.addTween(myBox, {x:400, time:4, transition:"linear"});
Tweener.addTween(myBox, {x:100, time:2, transition:"linear"});
// Tween is not overwritten anymore
Tweener.autoOverwrite = false;
Tweener.addTween(myBox, {x:400, time:4, transition:"linear"});
Tweener.addTween(myBox, {x:100, time:2, transition:"linear"});
