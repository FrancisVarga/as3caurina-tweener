Reference » Tweening Parameters

delay

Availability

AS2 and AS3.

Usage

... delay:value, ...

Parameters

value:Number — How much time the transition must wait before it starts, in seconds (or in frames, if useFrames is set to true). This is used to create delayed transitions or more complex sequential animation. The default value for this parameter (when omitted) is 0.

Examples

// Slides a MovieClip to _x = 200 after 10 seconds are elapsed (AS2)
Tweener.addTween(myMovieClip, {_x:200, time:1, delay:10});
// Fades a MovieClip in, then out (AS2)
Tweener.addTween(myMovieClip, {_alpha:100, time:1});
Tweener.addTween(myMovieClip, {_alpha:0, time:1, delay:1});
// Uses an empty tweening to trace "hey!" after 5 seconds
Tweener.addTween(this, {delay:5, onComplete:function() { trace("hey!"); }});
// Sends a MovieClip to _x = 200 *immediately* after 10 seconds are elapsed (AS2)
Tweener.addTween(myMovieClip, {_x:200, delay:10});
